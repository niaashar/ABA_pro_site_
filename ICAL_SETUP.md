# Автоматическое обновление встреч

Сайт использует календарь заказчика с идентификатором `36506515` и московским временем.

## Настройка

1. В Яндекс Календаре заказчика скопируйте приватную ссылку экспорта iCal.
2. На GitHub откройте **Settings → Secrets and variables → Actions**.
3. Создайте секрет `YANDEX_CALENDAR_ICS_URL` и вставьте в него приватную iCal-ссылку.
4. Откройте **Settings → Actions → General → Workflow permissions** и выберите **Read and write permissions**.
5. В разделе **Actions** запустите workflow **Обновить встречи из Яндекс Календаря** кнопкой **Run workflow**.

Автоматическая проверка выполняется примерно каждые 10 минут. GitHub может запускать задачу с небольшой задержкой.

## Шаблон события

Название: `Для родителей | Тема встречи` или `Для специалистов | Тема встречи`

Описание:

```
Для кого: Для родителей
Формат: Онлайн
Стоимость: 1 500 ₽
Описание: Краткое описание встречи.
```

Закрытую ссылку на подключение не размещайте в публичном календаре.
